# Data Science Repo Template

A repository template using Poetry, Makefile, and pre-commit-hooks

## Project structure

```bash
.
├── data
│   ├── processed
│   └── raw
├── deploy
│   ├── scripts
│   └── tests
├── develop
│   └── notebooks
├── label
├── train
├── visualise
├── .flake8
├── .gitattributes
├── .gitignore
├── .pre-commit-config.yaml
├── Makefile
├── poetry.toml
├── pyproject.toml
└── README.md
```
